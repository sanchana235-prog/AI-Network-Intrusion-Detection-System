# AI-Based Network Intrusion Detection System

## Overview

This project uses Machine Learning to detect malicious network traffic.

A Random Forest Classifier was trained on the NSL-KDD cybersecurity dataset to classify network connections as either:

- Normal Traffic
- Attack Traffic

## Features

- Intrusion Detection using Machine Learning
- Random Forest Classification
- Streamlit-based User Interface
- CSV Upload for Network Traffic Analysis
- Real-time Prediction of Normal vs Attack Traffic

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Jupyter Notebook

## Dataset

NSL-KDD Dataset

The dataset contains network traffic records with multiple features describing connection behavior.

## Methodology

1. Data Loading
2. Data Preprocessing
3. Label Encoding
4. Train-Test Split
5. Random Forest Training
6. Prediction and Evaluation

## Results

- Accuracy: 99.86%
